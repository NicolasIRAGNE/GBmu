enum gpu_mode_e

The current state of the PPU. These modes indicate what the PPU is currently doing and which memory areas are currently accessible.

Enumerator
GPU_MODE_HBLANK 

PPU is in HBLANK mode. All memory is accessible.

GPU_MODE_VBLANK 

PPU is in VBLANK mode. All memory is accessible. It's free real estate.

GPU_MODE_OAM 

PPU is reading from OAM. Access to these memory areas is blocked: OAM.

GPU_MODE_VRAM 

PPU is reading from VRAM and OAM. Access to these memory areas is blocked: VRAM, OAM and palettes.
